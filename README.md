# Project2MissionD

In addition to completing the tutorial, I enhanced the game by adding background music, a particle effect for the projectile, a sound effect for the projectile's first collision, and a sound effect when the projectile hits a target. These changes improve the game's feedback and make collisions feel more satisfying and noticeable.
